The maintainable IoT distribution

What is Geckito?

Geckito is an umbrella project to make (open)SUSE distributions ready for Edge and IoT environments.

Build Tooling

pbuild is able to build multiple artefacts in many formats like images, container or packages. It can work complete offline, but also use online resources.

A distribution in binary and source form.

The pbuild tool is distribution agnostic and supports rpm, debian and arch distributions. However the geckito project is focusing on the (open)SUSE distributions to deliver binaries and sources suitable for embedded development. This includes reproducible builds, cross architecture builds for a defined maintenance time frame.

A template collection

Templates are demonstrating how the tooling and distribution can be used to build maintainable images designed for IoT devices. Unlike classic Linux distribution images these images are supposed to get rebuild for updating the system. As a result the images can become much smaller.
